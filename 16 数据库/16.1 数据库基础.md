**设置mysql密码**

```
mysqladmin -u root password '123456'
```

注意：这是在未登录数据库的情况下使用。

**启动数据库**

```
mysql.server start
```

**关闭数据库**

```
mysql.server stop
```

**登录数据库**

```
mysql -u root -p
```

查看所有数据库

```
show databases;
```

切换数据库

```
use mysql;
```

显示当前数据库中所有表

```
show tables;
```

退出数据库

```
quit; 
-- 或者
exit;
```

增删改查

```
-- 创建数据库
create database mydb;
create database mytest charset=utf8;

-- 删除数据库
drop database mydb;
```
